# Weather App

A beautiful and feature-rich weather application built with Flutter that provides real-time weather information using a weather API. The app displays current weather conditions, hourly forecasts, and daily forecasts for any location worldwide.

## Features

- 🌤️ **Current Weather**: Get real-time temperature, humidity, wind speed, and weather conditions.
- 📅 **Hourly Forecast**: View weather predictions for the next 24 hours.
- 📆 **Daily Forecast**: 7-day weather forecast with min/max temperatures.
- 🔍 **Search Locations**: Search for weather in any city or town.
- 📍 **Current Location**: Automatically fetch weather for your current location (with permission).
- 🎨 **Clean UI**: Intuitive and responsive design for both Android and iOS.
- 🌓 **Dark Mode**: Supports light and dark themes based on system settings.

## API Used

This app uses the [OpenWeatherMap API](https://openweathermap.org/api) to fetch weather data. You will need to obtain a free API key from their website to run the app.
